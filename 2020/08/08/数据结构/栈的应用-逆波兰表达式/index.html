栈的应用-逆波兰计算器

  1. 中缀表达式：

     是常见的运算表达式如：(8+5) X 6 -7 ，不好被计算机操作，需要判断符号优先级

  2. 后缀表达式:

     也叫波兰表达式：(8+5) X 6 -7 写成逆波兰表达式 8 5 + 6 X 7 -
 我们平时写a+b，这是中缀表达式，写成后缀表达式就是：

 ab+
 (a+b)*c-(a+b)/e的后缀表达式为：
 (a+b)*c-(a+b)/e
 →((a+b)*c)((a+b)/e)-
 →((a+b)c*)((a+b)e/)-
 →(ab+c*)(ab+e/)-
 →ab+c*ab+e/-

  3. 中缀表达式转后缀表达式：

    1. 初始化两个栈：运算栈s1 和 结果栈s2

    2. 从左到右扫描中缀表达式

    3. 遇到数时，push s2

    4. 遇到运算符，比较 s1栈顶运算符的优先级：

       4.1  如果s1为空或栈顶元素为 “(” 则pushs1
 4.2  优先级比栈顶运算符高， 也push s1
 4.3 否则将s1栈顶元素pop出 并push到s2中 再回到4.1 与s1 比较 重复步骤

    5. 遇到括号时：

       5.1 左括号 “(” ，push s1
 5.2 右括号 “)" 则pop s1 并 push s2
 直到遇到 左括号 “(” ，将括号丢弃 重复 2 - 5 步骤

    6. 将s1 中的元素 push 到s2

    7. s2 pop即为中缀表达式对应的后缀表达式

    Stack<Character> operStack = new Stack<>();  //运算符栈
    Stack<Character> resStack = new Stack<>();   //结果栈
    for(int i=0; i<str.length(); i++)&#123;
        char ch = str.charAt(i);
        if(isOper(ch))&#123;  //符号判断
            boolean flag = true;
            if(ch == ')')&#123; //右括号
                while (flag)&#123;
                    if(operStack.peek() == '(')&#123;
                        operStack.pop();
                        flag = false;
                    &#125;else &#123;
                        resStack.push(operStack.pop());
                    &#125;
                &#125;
            &#125;else &#123;   //没遇到右括号
                while (flag) &#123;
                    if (operStack.size() == 0) &#123;
                        operStack.push(ch);
                        flag = false;
                    &#125;else if( operStack.peek() == '(' )&#123;
                        operStack.push(ch);
                        flag = false;
                    &#125;else if (getPriority(ch) > getPriority(operStack.peek())) &#123;
                        operStack.push(ch);
                        flag = false;
                    &#125; else &#123;
                        resStack.push(operStack.pop());
                    &#125;
                &#125;
            &#125;
        &#125;else &#123;   //数字

        &#125;
    &#125;
    resStack.push(operStack.pop());
    StringBuilder builder = new StringBuilder("");
    while (resStack.size() != 0)&#123;
        //逆向输出
        operStack.push(resStack.pop());

/**
 * 计算
 * @return

private static int cal(List<String> list)&#123;
    Stack<Integer> numberStack = new Stack<>();
    for(String item : list)&#123;
        if(item.matches("\\d+"))&#123; //多位数
            numberStack.push(Integer.parseInt(item));
        &#125;else&#123;
            int NumberTop = numberStack.pop();
            int NumberTopNext = numberStack.pop();
            int res = 0;
            switch (item)&#123;
                case "+":
                    res = NumberTop + NumberTopNext;
                    break;
                case "-":
                    res = NumberTopNext - NumberTop;
                    break;
                case "*":
                    res = NumberTop * NumberTopNext;
                    break;
                case "/":
                    res = NumberTopNext / NumberTop;
                    break;
                    default:
                        throw new RuntimeException("未知错误");
            &#125;
            numberStack.push(res);
        &#125;
    &#125;
    return numberStack.pop();
